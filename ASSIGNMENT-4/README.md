# IN226060302_FASTAPI – Assignment - 4

### 📌 Project Description

This project implements a **Shopping Cart System API using FastAPI**.
It allows users to add products to a cart, view the cart, remove items, and checkout orders.
The API simulates a basic **e-commerce workflow** where multiple customers can interact with the cart system and place orders.

## 🚀 Features

* Add products to cart
* View cart items
* Update quantity when the same product is added again
* Remove products from cart
* Checkout cart items
* View orders placed
* Error handling for:

  * Out-of-stock products
  * Invalid product IDs
  * Checkout with empty cart


## 🛠 Technologies Used

* **Python**
* **FastAPI**
* **Uvicorn**
* **Pydantic**
